# Brand Style Notes

Brand qualities:

- Bold.
- Intelligent.
- Strategic.
- Human.
- Operator-room first.
- Editorial, dense, receipt-backed, and practical.

Visual language:

- Deep green/ink foundation.
- Warm parchment surface.
- Gold signal accents.
- Editorial vector illustrations.
- Strong diagrams rather than generic SaaS decoration.

Typography direction:

- Clear editorial headings.
- Dense but readable body copy.
- Labels and status chips should support scanning.
