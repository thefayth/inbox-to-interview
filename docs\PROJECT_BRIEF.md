# Project Brief

Inbox To Interview is a local-first command center for turning job-search inbox chaos into a receipt-backed workflow.

It coordinates role intake, triage, resume packet creation, application kits, approvals, recruiter replies, ATS/LinkedIn assist, follow-ups, interview prep, and reporting.

The public repository explains the project and protects the private operating system.

Core position: public surface, private engine, receipts always.
